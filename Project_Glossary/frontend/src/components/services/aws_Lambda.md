# AWS Lambda

サーバーレスコンピューティングサービス。サーバーの管理なしでコードを実行できます。

## 主な特徴
- サーバー管理不要
- 使用した分だけ課金
- 自動スケーリング
- 多言語対応（Python、Node.js、Java等）

## 主な機能
- イベント駆動型アーキテクチャ

    Lambda関数は、様々なAWSサービスからのイベントをトリガーとして実行。
    S3バケットへのオブジェクトのアップロード、DynamoDBテーブルへのデータの書き込み、API Gatewayへのリクエストなどをトリガーにできる

- 自動スケーリング

    Lambdaは、受信したイベント数に応じて、自動的にリソースをスケーリングする。

- 多様なランタイムのサポート

    Python, Node.js, Java, Go, Ruby, .NETなどの様々なプログラミング言語をサポート

- バージョン管理

    Lambda関数のバージョンを管理できる。 これにより、関数を安全に更新し、必要に応じて以前のバージョンに戻すことができる。

- 統合性と拡張性

    AWS Lambdaは、S3, DynamoDB, API Gateway, SNS, SQS, KinesisなどのAWSの他のサービスと密接に統合されており、様々なアプリケーションを構築できる。
    APIを利用して、Lambda関数の管理を自動化することも可能。
