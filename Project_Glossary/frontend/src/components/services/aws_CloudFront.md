# Amazon CloudFront

グローバルなコンテンツ配信ネットワーク（CDN）サービスです。

## 主な特徴
- 世界中のエッジロケーション
- 低レイテンシー配信
- DDoS攻撃保護
- SSL/TLS暗号化

## 主な機能
- S3、EC2、オンプレミスなどのサーバーに格納されているコンテンツを配信
    
    エッジロケーションでキャッシュして配信を高速化するWebサービス

- コンテンツの地理的ディストリビューションの制限

    地理的制限を使用すると、CloudFrontディストリビューションを通じて配信しているコンテンツに対して、特定地域のユーザーがアクセスアクセス出来ないようにする事ができる

- OAIの設定で

    ユーザーはS3バケットから直接ではなく、CloudFront経由でのみファイルにアクセスできるようにするための特別なユーザー
